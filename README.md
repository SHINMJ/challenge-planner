# Challenge Planner API

“Push Up 100개 30일 도전하기” 와 같은 나만의 챌린지를 관리하고 같이 할 사람과 공유할 수 있는 서비스.

---

클린 아키텍처의 개념을 활용하여 아키텍처를 만들고(?) Spring WebFlux를 사용하여 비동기 논블로킹 방식으로 API를 처리한다.

---

## 개발 환경
- spring boot 3
- gradle 8
- jdk 17
- spring webflux
